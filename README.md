# Steno28
A simple stenography keyboard with 28 keys. I made this keyboard because I find Stenography keboards interesting and I decided to try making one myself. The PCB is relatively simple, being designed in KiCAD and using the Raspberry Pi Pico RP2040 to connect to the computer. <img width="1448" height="467" alt="Screenshot 2026-05-06 at 7 28 45 AM" src="https://github.com/user-attachments/assets/910ccc35-e178-40aa-955a-736ee8e19a39" /> I didn't bother attempting onboard programming, because that would require attempting to code every single chord I would ever want to type. Rather, I will be using Plover (https://opensteno.org/plover/) that will translate the chords into words. The case is pretty simple, with the PCB just directly screwed on to the case, making it seem as if theres almost no case. Its really just there to protect the wiring. It will look something like this (sry about the lack of keys...) <img width="1374" height="600" alt="Screenshot 2026-05-06 at 7 42 40 AM" src="https://github.com/user-attachments/assets/fd3ce54d-aa8f-4a7e-ba9c-ec46f6abba98" /> I designed the case in Onshape:https://cad.onshape.com/documents/53637eff7d4d43c451aa77bb/w/a1ff151550c121314276b672/e/7c0bff2a6c8d979b5125e58d?renderMode=2&uiState=69fb53bf7ce38d7c472fc839
